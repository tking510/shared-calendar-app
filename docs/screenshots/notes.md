# スクリーンショット撮影メモ

## 1. ログイン画面
- URL: /login
- 内容: ユーザー名・パスワード入力、ログインボタン、新規登録リンク

## 2. カレンダー画面（ホーム）
- URL: /
- 内容: 月間カレンダー表示、日付選択、予定一覧、+ボタン
- 特徴: 今日の日付がハイライト、予定のある日にドット表示

## 3. 共有タブ
- URL: / (共有タブ)
- 内容: 共有カレンダー一覧、作成・参加ボタン

## 4. 設定タブ
- URL: / (設定タブ)
- 内容: タグ管理、Telegram設定、友達管理、部署管理、ログアウト

## 5. 予定作成画面
- URL: /event/new
- 内容: タイトル、日時、場所、タグ、リマインダー設定

## 6. 予定詳細画面
- URL: /event/[id]
- 内容: 予定の詳細表示、編集・削除ボタン
