## 1. はじめに

### アプリの画面構成

アプリは下部のタブで3つの画面に切り替えられます。

| タブ | アイコン | 機能 |
|------|---------|------|
| **カレンダー** | 📅 | 予定の確認・作成 |
| **共有** | 👥 | 共有カレンダーの管理 |
| **設定** | ⚙️ | タグ・通知・アカウント設定 |

---

## 2. ログイン・アカウント登録

### 新規登録の手順

```
① 「アカウント登録」をタップ
② ユーザー名を入力（半角英数字）
③ パスワードを入力（6文字以上）
④ 表示名を入力（任意）
⑤ 「登録」をタップ
```

### ログインの手順

```
① ユーザー名を入力
② パスワードを入力
③ 「ログイン」をタップ
```

> **ヒント**: ログイン状態は保持されるため、毎回入力する必要はありません。

## 3. カレンダーの基本操作

### 月の移動

| 操作 | 方法 |
|------|------|
| 前の月へ | 左の **＜** ボタンをタップ |
| 次の月へ | 右の **＞** ボタンをタップ |
| 今日に戻る | **「今日」** ボタンをタップ |

### 日付の選択

カレンダーの日付をタップすると、その日の予定一覧が下に表示されます。

### 予定の色の意味

| 表示 | 意味 |
|------|------|
| 🔴 赤い丸 | 今日の日付 |
| 薄い背景色 | 選択中の日付 |
| 小さなドット | 予定がある日 |

## 4. 予定を作成する

### 作成手順

```
① カレンダー画面右下の ＋ ボタンをタップ
② タイトルを入力（必須）
③ 開始日時・終了日時を設定
④ 必要に応じてオプションを設定
⑤ 右上の「保存」をタップ
```

### 設定できる項目

| 項目 | 説明 | 必須 |
|------|------|------|
| **タイトル** | 予定の名前 | ✅ |
| **開始日時** | 予定の開始時刻 | ✅ |
| **終了日時** | 予定の終了時刻 | ✅ |
| **場所** | 場所やURLなど | - |
| **メモ** | 詳細な説明 | - |
| **繰り返し** | 毎日/毎週/毎月/毎年 | - |
| **タグ** | 分類用のラベル | - |
| **リマインダー** | 通知タイミング | - |
| **友達** | 通知する相手 | - |
| **部署** | 通知するグループ | - |
| **カスタムメッセージ** | 通知に追加するメッセージ | - |

### 繰り返し設定

| 設定 | 動作 |
|------|------|
| **なし** | 1回のみの予定 |
| **毎日** | 毎日同じ時間に表示 |
| **毎週** | 毎週同じ曜日に表示 |
| **毎月** | 毎月同じ日に表示 |
| **毎年** | 毎年同じ日に表示 |

### リマインダー設定

予定の前に通知を受け取れます。複数選択可能です。

| 選択肢 |
|--------|
| 5分前 |
| 15分前 |
| 30分前 |
| 1時間前 |
| 1日前 |

## 5. 予定を編集・削除する

### 編集手順

```
① カレンダーで予定をタップ → 詳細画面へ
② 右上の「編集」をタップ
③ 内容を変更
④ 「保存」をタップ
```

### 削除手順

```
① カレンダーで予定をタップ → 詳細画面へ
② 下部の「削除」ボタンをタップ
③ 確認ダイアログで「削除」を選択
```

> **注意**: 削除した予定は元に戻せません。

## 6. 共有カレンダーを使う

共有カレンダーを使うと、チームや家族と予定を共有できます。

### 共有カレンダーを作成する

```
① 「共有」タブをタップ
② 右上の ＋ ボタンをタップ
③ カレンダー名を入力
④ 「作成」をタップ
⑤ 表示された招待コード（8文字）を相手に伝える
```

### 共有カレンダーに参加する

```
① 「共有」タブをタップ
② 「参加」ボタンをタップ
③ 招待コード（8文字）を入力
④ 「参加」をタップ
```

### 共有カレンダーに予定を追加する

```
① 「共有」タブで対象のカレンダーをタップ
② 「予定を追加」ボタンをタップ
③ 予定を入力して保存
```

> **ポイント**: 共有カレンダーに追加した予定は、メンバー全員が見ることができます。

### メンバーの役割

| 役割 | できること |
|------|-----------|
| **オーナー** | 予定の追加・編集・削除、招待コードの確認 |
| **編集者** | 予定の追加・編集・削除 |
| **閲覧者** | 予定の閲覧のみ |

## 7. タグで予定を整理する

タグを使って予定を分類できます。

### タグを作成する

```
① 「設定」タブをタップ
② 「タグ管理」セクションの ＋ をタップ
③ タグ名を入力
④ 色を選択
⑤ 「保存」をタップ
```

### タグの使い方の例

| タグ名 | 色 | 用途 |
|--------|-----|------|
| 仕事 | 🔵 青 | 業務関連の予定 |
| プライベート | 🟢 緑 | 個人的な予定 |
| 重要 | 🔴 赤 | 優先度の高い予定 |
| 家族 | 🟠 オレンジ | 家族との予定 |

## 8. Telegram通知を設定する

予定のリマインダーをTelegramで受け取れます。

### 事前準備

1. Telegramアプリをインストール
2. **@BotFather** で自分のBotを作成してBot Tokenを取得
3. 自分のChat IDを確認（@userinfobot などで確認可能）

### 設定手順

```
① 「設定」タブをタップ
② 「Telegram通知設定」をタップ
③ Bot Tokenを入力
④ Chat IDを入力
⑤ スレッドID（任意）を入力
⑥ 「テスト通知を送信」で動作確認
⑦ 「保存」をタップ
```

### 個人のTelegram通知設定

自分だけに通知を受け取りたい場合は、「個人Telegram設定」で自分のChat IDを設定します。

```
① 「設定」タブをタップ
② 「個人Telegram設定」をタップ
③ 自分のTelegram Chat IDを入力
④ 「保存」をタップ
```

## 9. 友達・部署の管理

予定に友達や部署をタグ付けすると、その人たちにもTelegram通知が届きます。

### 友達を追加する

```
① 「設定」タブをタップ
② 「友達管理」の ＋ をタップ
③ 名前を入力
④ Telegram Chat ID または ユーザー名を入力
⑤ 色を選択
⑥ 「保存」をタップ
```

### 部署を管理する（管理者のみ）

```
① 「設定」タブをタップ
② 「管理者モード」をONにする
③ 管理者パスワードを入力
④ 「部署管理」の ＋ をタップ
⑤ 部署名と色を設定
⑥ 「保存」をタップ
```

> **管理者パスワード**: システム管理者から取得してください。

### 自分の所属部署を設定する

```
① 「設定」タブをタップ
② 「自分の部署」をタップ
③ 所属する部署にチェック
④ 「保存」をタップ
```

## 10. よくある質問

### Q: 予定を作成したのに表示されない

**A**: 画面を下に引っ張って更新（プルリフレッシュ）してください。

### Q: 共有カレンダーの予定が見えない

**A**: 共有カレンダー詳細画面から「予定を追加」ボタンで予定を作成してください。メインのカレンダー画面から作成した予定は個人の予定になります。

### Q: Telegram通知が届かない

**A**: 以下を確認してください。

| チェック項目 |
|-------------|
| Bot Tokenが正しいか |
| Chat IDが正しいか |
| Botをブロックしていないか |
| 「テスト通知を送信」で届くか |

### Q: 繰り返し予定を1回だけスキップしたい

**A**: 現在、個別のスキップ機能はありません。繰り返しを解除して、必要な日だけ個別に予定を作成してください。

### Q: パスワードを忘れた

**A**: システム管理者にお問い合わせください。
